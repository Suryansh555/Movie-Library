Movie Library

The project uses - HTML, CSS, JavaScript, Jquery and Firebase. 

We have used authentication service using Firebase. We can read, write as well as update using this. 
Features given for the application is - 
1. Sign up
2. Login
3. Log Out

Currently, we have two screens post authentication. They are - 
1. Home
Here, we can search for any movie using the omdbapi. On searching we get an option to view movie details or to add to the list. 
[The list feature is Work In Progress]

2. Movie
On this page we get the details on the movie. We can also be direted to imdb page to view further. There are navigation option through 
nav bar or the button in the end to go back to home screen. 

The Web App can run directly by running the index.html file in any web browser.
